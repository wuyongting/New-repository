# 抽獎轉盤應用

這是一個簡單的抽獎轉盤應用，包含三個選項：1000、1、100。

## 部署方式

### 方式一：GitHub Pages（最簡單，免費）

1. **建立 GitHub 帳號**（如果還沒有的話）
   - 前往 https://github.com 註冊

2. **建立新專案**
   - 點擊右上角 "+" → "New repository"
   - 輸入專案名稱（例如：lottery-wheel）
   - 選擇 Public（公開）
   - 點擊 "Create repository"

3. **上傳檔案**
   - 在專案頁面，點擊 "uploading an existing file"
   - 把這三個檔案拖進去：`index.html`、`style.css`、`script.js`
   - 點擊 "Commit changes"

4. **開啟 GitHub Pages**
   - 點擊 Settings（設定）
   - 左側選單找到 "Pages"
   - Source 選擇 "main" 分支
   - 點擊 Save
   - 等待幾分鐘，會出現網址：`https://你的帳號.github.io/專案名稱/`

### 方式二：Vercel（也很簡單，免費）

1. **建立 Vercel 帳號**
   - 前往 https://vercel.com
   - 用 GitHub 帳號登入（最簡單）

2. **部署**
   - 點擊 "Add New Project"
   - 選擇你的 GitHub 專案（或直接上傳檔案）
   - 點擊 "Deploy"
   - 完成後會給你一個網址

### 方式三：Netlify（同樣簡單，免費）

1. **建立 Netlify 帳號**
   - 前往 https://www.netlify.com
   - 用 GitHub 帳號登入

2. **部署**
   - 把整個資料夾拖到 Netlify 頁面
   - 自動部署完成
   - 會給你一個網址

## 檔案說明

- `index.html` - 主頁面
- `style.css` - 樣式檔案
- `script.js` - 功能程式碼

## 注意事項

- 記錄功能目前已經隱藏
- 這是純前端應用，不需要伺服器
- 可以直接在瀏覽器打開 `index.html` 測試
